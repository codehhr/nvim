# 我的 `nvim` 配置

![nvim](https://codehhr.coding.net/p/codehhr/d/images/git/raw/master/vim/vim-screenshot.png)

没有仔细配置 , 需要安装 `nodejs` , `ctags`

## 安装后需要

```sh
npm install -g neovim
```
or
```sh
sudo npm install -g neovim
```
